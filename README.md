# Jail-Management-System

## Functional Requirements

### Admin: Warden

- **Update Personal Details:**
  - The system allows the admin to update the personal details of every staff/other user within the system.

- **Add New Staff/Users:**
  - The system allows the admin to add the personal details of new staff/new users within the system.

- **Modify Working Schedules:**
  - The system offers the admin to modify the working schedules of every staff/other user in the system.

- **Delete Staff Information:**
  - This system helps the admin to delete the information regarding the staff.

- **Modify Prisoner Details:**
  - This system offers the admin to modify the details regarding the prisoners.

- **View Staff Counts:**
  - This system offers the admin to see the counts of all staff, including the prisoners.

### Users: Deputy Warden

- **Assign Work Type and Shift:**
  - This system allows the deputy warden to assign the new users' work type and shift.

### Police Module

- **View Working Schedules:**
  - This system permits the users who are 'Police' to see their working schedules in the jail.

- **Request Schedule Change:**
  - This system allows the 'Police' users to request a change in their working schedule.

### Chef Module

- **View Working Schedules:**
  - This system permits the users who are 'Chef' to see their working schedules in the jail.

- **Request Schedule Change:**
  - This system allows the 'Chef' users to request a change in their working schedule.

### Cleaner Module

- **View Working Schedules:**
  - This system permits the users who are 'Cleaner' to see their working schedules in the jail.

- **Request Schedule Change:**
  - This system allows the 'Cleaner' users to request a change in their working schedule.
